## Бутик

### Карточка товара

#### Копирование Product Code

Статус: Отложено

Описание:
Добавить возможность копирования Product Code в буфер обмена по нажатию.

Причина:
Сейчас в этом нет необходимости.
Покупатель либо сразу пишет менеджеру в Telegram, либо использует кнопку «Поделиться», которая уже содержит информацию о товаре.

Вернуться к идее только при появлении реальной потребности.
